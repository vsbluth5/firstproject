Arrays vs. Array Lists

Both arrays and array lists are data structures which store elements in sequentially in memory. This means that the element sitting at index 5 is stored between the elements at index 4 and the one at index 6

The major differences between an array and an array list:

  • How the elements are accessed and assigned

    Arrays are accessed using the index. Array lists are accessed using the get command.
